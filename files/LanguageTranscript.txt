Why use batch transcription?
Batch transcription is ideal if you want to transcribe a large quantity of audio and storage such as Azure blobs.
By using the dedicated rest API, you can point to audio files with a shared access signature Uri in asynchronously receive transcriptions.
As with all features of the speech service, you create a subscription key from the Azure Portal by following our get started guide.
If you plan to get transcriptions from our baseline models, creating a key is all you need to do.
